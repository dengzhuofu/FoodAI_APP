# Recommended Python AI Libraries

## Core LLM SDKs
- **openai**: Official client for OpenAI API.
- **anthropic**: Official client for Claude/Anthropic API.
- **google-generativeai**: Client for Gemini models.

## Orchestration & Frameworks
- **langchain**: Comprehensive framework for building LLM apps. Good for complex chains.
- **langchain-core**: Core abstractions, lighter weight.
- **langchain-community**: Community integrations.
- **langgraph**: For building stateful, multi-actor applications (agents).
- **llamaindex**: Data framework for connecting custom data sources to LLMs.

## Web UI & API
- **streamlit**: Pure Python way to create web apps. Great for data apps and prototypes.
- **gradio**: Easy UI for ML models.
- **chainlit**: Python package to build production ready Chat UI in minutes.
- **fastapi**: Modern, fast (high-performance) web framework for building APIs with Python.

## Utilities
- **python-dotenv**: Loads environment variables from .env file.
- **pydantic**: Data validation using Python type hints. Essential for structured output.
- **tenacity**: Retrying library, useful for handling API rate limits and errors.
- **tiktoken**: Tokenizer for OpenAI models.

## Vector Databases (Local)
- **chromadb**: Simple, open-source embedding database.
- **faiss-cpu**: Efficient similarity search and clustering of dense vectors.
- **lancedb**: Developer-friendly, serverless vector database.
