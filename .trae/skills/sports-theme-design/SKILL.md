# Sports Theme Design System

## Core Philosophy
Transform the food app into an energetic, competition-based experience.
- **Keywords**: Speed, Energy, Competition, Bold, Vibrant.
- **Visuals**: High contrast, rounded cards, gradient accents.

## Color Palette

### Primary Colors
- **Teal Green**: `#00C896` (Main Brand Color - Buttons, Accents, Active States)
- **Deep Teal**: `#00A87E` (Pressed states, Darker accents)

### Secondary Colors
- **Neon Yellow**: `#EBFF00` (Highlights, Badges, "New" tags)
- **Electric Blue**: `#2DCCFF` (Secondary actions, Information)

### Backgrounds
- **Dark Mode (Default)**: `#121212` (Main Background)
- **Card Background (Dark)**: `#1E1E1E` (Cards on dark bg)
- **Card Background (Light)**: `#FFFFFF` (Cards on light bg - if applicable)
- **Surface**: `#252525` (Modals, Bottom Sheets)

### Text
- **Primary**: `#FFFFFF` (On dark), `#121212` (On light)
- **Secondary**: `#A0A0A0` (Subtitles)
- **Accent**: `#00C896` (Links, Active text)

## Typography

### Headings
- **Font Family**: System Bold / Heavy (San Francisco, Roboto)
- **H1**: 32px, Bold, Italic (Optional for "Speed" look)
- **H2**: 24px, Bold
- **H3**: 20px, SemiBold

### Body
- **Body Regular**: 16px, Regular
- **Body Small**: 14px, Regular
- **Caption**: 12px, Medium, Uppercase

## Spacing & Layout
- **Grid**: 4px baseline.
- **Padding**: Standard container padding `20px`.
- **Gap**: Standard element gap `12px` or `16px`.
- **Radius**:
    - **Cards**: `16px` or `20px` (Soft, large curves)
    - **Buttons**: `12px` or `Full Round` (Pill shape)

## Components

### Buttons
- **Primary**: Solid Teal `#00C896`, White Text, Rounded `12px`.
- **Secondary**: Transparent with Teal Border `2px`.
- **Ghost**: Text only, Teal color.

### Cards
- **Style**: Elevated, Soft Shadow.
- **Shadow**: `0px 4px 12px rgba(0, 200, 150, 0.2)` (Teal glow).

### Tabs
- **Active**: Teal Text + Teal Bottom Bar (or Pill indicator).
- **Inactive**: Gray Text.
