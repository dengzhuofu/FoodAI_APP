接下来的开发工作将分为后端数据库初始化和前端认证模块对接两个主要阶段，旨在打通前后端联调的基础链路。

## 1. 后端数据库初始化 (Backend Setup)
*   **目标**: 确保数据库表结构正确创建，后端服务可运行。
*   **操作**:
    1.  安装后端依赖 (`pip install -r backend/requirements.txt`)。
    2.  使用 Aerich 初始化数据库配置 (`aerich init -t app.core.config.settings.TORTOISE_ORM`)。
    3.  执行数据库迁移和升级 (`aerich init-db`)，在 MySQL 中创建表结构。

## 2. 前端环境配置 (Frontend Setup)
*   **目标**: 引入网络请求和本地存储能力。
*   **操作**:
    1.  安装 `axios` (用于 API 请求)。
    2.  安装 `expo-secure-store` (用于安全存储 JWT Token)。

## 3. 前端 API 层封装 (API Integration)
*   **目标**: 统一管理 API 请求，实现自动 Token 注入。
*   **文件**:
    1.  `src/api/client.ts`: 创建 Axios 实例，配置拦截器自动附加 Token。
    2.  `src/api/auth.ts`: 封装 `login` (登录) 和 `register` (注册) 接口调用。

## 4. 前端认证页面开发 (UI Implementation)
*   **目标**: 实现用户登录和注册界面，并连接后端接口。
*   **文件**:
    1.  `src/app/screens/Auth/LoginScreen.tsx`: 登录页面，包含表单验证和 API 调用。
    2.  `src/app/screens/Auth/RegisterScreen.tsx`: 注册页面。
    3.  `src/app/navigation/AppNavigator.tsx`: 更新导航结构，加入认证流程（AuthStack），默认进入登录页，成功后跳转主页。
